Class BpmnStyleIdentifier

Define BPMN specific keys used in mxGraph styles. Use constants defined in this class instead of hard coded string values.

experimental

You may use this to customize the BPMN theme as proposed in the examples. But be aware that the way we store and allow to change the defaults is subject to change.
