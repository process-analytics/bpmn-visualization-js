Class BpmnCanvas

Wrapper of {@link mxAbstractCanvas2D} to simplify method calls when painting icons/markers of BPMN shapes.

It can scale dimensions passed to the method of the original {@link mxAbstractCanvas2D}

example

vanilla canvas calls when a scale factor must be applied to positions const scaleX = 0.26; const scaleY = 0.35; c.moveTo(8 * scaleX, 39 * scaleY); c.lineTo(12 * scaleX, 25 * scaleY);

example

with BpmnCanvas const canvas = new BpmnCanvas(c, 0.26, 0.35); canvas.moveTo(8, 39); canvas.lineTo(12, 25);

experimental

You may use this to customize the BPMN theme as proposed in the examples. But be aware that the way we store and allow to change the defaults is subject to change.
