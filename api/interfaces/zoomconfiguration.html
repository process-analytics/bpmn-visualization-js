Optional debounceDelay

debounceDelay: number
  • Defined in component/options.ts:59

debounceDelay [ms] responsible for debouncing the zoom function - the actual scaling. A bigger value results in bigger gain in zoom factor before actual scaling takes place. Values must be in the [0, 100] interval, values outside of this interval are set to the interval bounds.

default

50

Optional throttleDelay

throttleDelay: number
  • Defined in component/options.ts:53

throttleDelay [ms] responsible for throttling the mouse scroll event (not every event is firing the function handler, only limited number can lunch handler). A smaller value results in more events fired, bigger gain in zoom factor. Values must be in the [0, 100] interval, values outside of this interval are set to the interval bounds.

default

50
